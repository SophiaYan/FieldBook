# FieldBook
FieldBook is an online platform helping intelligent farm management. This project is done by a team of four in 36 hours during HackIllinois 2016 and won the 3rd place in the Climate's Corp.'s competition among 26 teams.  
- Implemented Login functionality and retrieved user data using Climate Corp.’s API with security guaranteed.  
-	Crawled and integrated thousands of agricultural data from federal public databases.  
- Rated and visualized user’s farm using metadata, Google API and D3.js
